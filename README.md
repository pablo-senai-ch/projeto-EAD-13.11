O meu projeto foi dividido em três partes principais: o HTML, o CSS e o JavaScript.
No HTML, construí a estrutura base do portfólio, formada por três grandes seções: header, main e footer.
O CSS foi utilizado para estilizar o site, trazendo cores, fontes, tamanhos e formatos que deram “vida” ao projeto.
Já o JavaScript foi responsável por adicionar dinamicidade e interatividade, tornando o portfólio mais moderno e envolvente para o usuário.
